The optimisation runs were done on a computer with the following specifications:
\begin{itemize}
\item CPU: Intel Core i7-6700K CPU @ 4.2GHz
\item Kernel: x86\_64 Linux 4.12.10-1-ARCH
\item RAM: 15973MiB
\end{itemize}
